# CareerNest – Full Stack Job Portal

CareerNest is a modern full-stack job portal platform that connects job seekers with recruiters. The platform allows companies to post and manage jobs while enabling candidates to search, apply, and track their applications through an intuitive interface.

## Features

### For Recruiters

* Create and manage job postings
* View and manage candidate applications
* Company profile management
* Dashboard for job and applicant tracking

### For Job Seekers

* Search and filter jobs
* Apply for jobs with resume upload
* Track application status
* Manage profile information

### General Features

* Secure authentication with Clerk
* Cloud-based file storage using Cloudinary
* Responsive user interface
* RESTful API architecture
* Error monitoring with Sentry

## Installation

### Clone the repository

```bash
git clone https://github.com/lakshmijahnavik/CareerNest.git
```

### Install dependencies

#### Client

```bash
cd client
npm install
```

#### Server

```bash
cd server
npm install
```

### Run the application

#### Backend

```bash
npm run server
```

#### Frontend

```bash
npm run dev
```
